# Criação de chatbot com IA
Este projeto consiste em um chatbot com Inteligência Artificial desenvolvido em Python utilizando Streamlit. A aplicação permite que usuários enviem mensagens em tempo real e recebam respostas geradas por IA através da API da OpenAI. O sistema mantém o histórico da conversa durante a sessão, criando uma experiência interativa semelhante a um chat moderno.

# Stacks utilizadas

O projeto foi desenvolvido utilizando a linguagem Python com foco em criação de aplicações web e integração com Inteligência Artificial. O framework Streamlit foi utilizado para construção da interface interativa do chatbot, enquanto a biblioteca OpenAI Python Library foi responsável pela comunicação com os modelos de IA da OpenAI. O sistema também utiliza gerenciamento de estado através do session_state do Streamlit para armazenar e exibir o histórico das conversas em tempo real.
